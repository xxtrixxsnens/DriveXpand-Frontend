# DriveXpand - Frontend

## Ideen
- Zeitenansicht (z.B Monatsansicht)
    - Aggregrierte Daten für ein gewissen Zeitpunkt
    - Durchschnittsgeschwindigkeit
    - Gefahrene Strecke
    - Fun Facts zu Aggregrierten Daten
        - "Du bist 1000km gefahren, dass ist soweit wie von Paris nach Berlin"

- Zu welchen Zeiten fährt man?
    - Wochentagen
    - Uhrzeiten
    - Jahreszeiten?

- Achievements
- Warnungen

- Fahrten
    - Wann man gefahren ist?
    - Wie viel man gefahren ist?
    - Dauer
    - Durchschnittsgeschwindigkeit

    - Fahrtenbuch
        - Einzelne Fahrten
        - Eintragen von Fahrten 
        -> Mehr Kontext e.g. Geburtstagsfeier
        -> Zu welchen Orten man gefahren ist?

- Anlegen von verschiedenen Autos
     - Anleitung, um die Daten vom Auto zum Server zu bekommen
     -> e.g einsetzen des Microcontrollers
